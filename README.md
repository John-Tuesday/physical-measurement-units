# physical-measurement-units

Simple Kotlin Multiplatform library which supports different types of units of measure and

maven coordinates: `io.github.john-tuesday:measure:$version`

---

### Measurement Dimensions
* Mass
* Volume
* Length

### Mass units
* micrograms
* milligrams
* grams
* kilograms
* pounds
* ounces

### Volume units
* Milliliters
* Liters
* Kiloliters
* US Gallons
* US Tablespoons
* US Teaspoons

### Length Units
* Millimeter
* Centimeter
* Meter
* Kilometer
* Inches
* Feet
* Miles
